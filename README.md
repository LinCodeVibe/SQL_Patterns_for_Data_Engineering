# Core SQL Patterns

### 1. Basic Queries

**Concepts:**

* SELECT
* WHERE
* ORDER BY

Used for simple data retrieval and filtering.

---

### 2. Aggregation ***

**Concepts:**

* COUNT()
* SUM()
* AVG()
* GROUP BY

Used for summarizing and analyzing data.

**Example Problem:**

* Department Highest Salary

---

### 3. JOINs ***

**Concepts:**

* INNER JOIN
* LEFT JOIN

Used to combine data from multiple tables.

**Example Problem:**

* Combine Two Tables

---

### 4. Subqueries ***

**Concepts:**

* Nested SELECT statements
* Correlated subqueries

Used when a query depends on another query.

**Example Problem:**

* Second Highest Salary

---

### 5. Window Functions ***

**Concepts:**

* ROW_NUMBER()
* RANK()
* DENSE_RANK()

Used for ranking, partitioning, and advanced analytics.

**Example Problem:**

* Rank Scores

### 6. Advanced SQL Logic

**Concepts:**

* CASE WHEN
